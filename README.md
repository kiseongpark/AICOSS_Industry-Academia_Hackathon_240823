# AICOSS Industry-Academia Hackathon

## 프로젝트 개요

AICOSS 산학연계 해커톤에서 진행한 프로젝트로, 기업 데이터를 기반으로 경영 개선에 필요한 주요 원인을 분석하고, 생성형 AI를 활용해 경영 개선 리포트를 자동 생성하는 것을 목표로 하였습니다.

특히 매출, 원재료비, 재고 등의 데이터를 분석하여 비용 증가 요인을 파악하고, 기업이 활용할 수 있는 개선 방향을 도출하는 데 초점을 맞추었습니다.

## 사용 기술

- Python
- Google Colab
- Pandas
- Matplotlib
- 생성형 AI API
- 데이터 전처리
- 이상치 분석
- 경영 데이터 분석
